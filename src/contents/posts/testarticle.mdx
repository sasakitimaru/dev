# リレーショナルデータベースのGo言語を用いた操作方法

Go言語は、シンプルで効率的なプログラミング言語として知られています。この記事では、Go言語を使用してリレーショナルデータベースを操作する基本的な方法を紹介します。

## データベース接続

Go言語でデータベースに接続するには、`database/sql`パッケージを使用します。

```go:go
import (
    "database/sql"
    _ "github.com/go-sql-driver/mysql"
)

func main() {
    db, err := sql.Open("mysql", "user:password@/dbname")
    // エラーハンドリング
}
```
## CRUD操作
### create操作
```go:go
_, err := db.Exec("INSERT INTO users(name, age) VALUES(?, ?)", "Alice", 25)
```
### read操作
```go
rows, err := db.Query("SELECT name, age FROM users")
```
## その他の便利な機能

- **トランザクション**: `db.Begin()`を使用してトランザクションを開始できます。
- **プリペアードステートメント**: SQLインジェクションを防ぐために使用します。

## まとめ

Go言語は、リレーショナルデータベースの操作に関しても非常に強力です。適切なパッケージとメソッドを使用することで、効率的かつ安全にデータベース操作を行うことができます。
